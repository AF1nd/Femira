# Femira

A simple programming language that compiles into vm bytecode

## Language basics:

":" - start block |
"end" - end block

"output" - print operator

define function:

```
def sum(arg1, arg2):
   return arg1 + arg2
end

output sum(1, 2)
```

if statement:

```
if 1 == 1:
   ..code
end
else:
   ..code
end
```
